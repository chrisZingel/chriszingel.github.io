# Minimal Mistakes remote theme starter

To setup a project based similar to this website then 
 [**Use this template**](https://github.com/mmistakes/mm-github-pages-starter/generate) button above for the quickest method of getting started with the [Minimal Mistakes Jekyll theme](https://github.com/mmistakes/minimal-mistakes).

## Run locally 
To allow local modifications while being able to see those changes
reflected live run

```bash
bundle exec jekyll serve -l
```
There are two parts here: first, the *bundle exec command*, which limits the gems to those specified in the Gemfile; and second, the jekyll serve command with the live reload flag enabled.
